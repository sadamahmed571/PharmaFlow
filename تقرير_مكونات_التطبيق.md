# تقرير مكونات و أقسام تطبيق صيدلية يمان (PharmaFlow)

## نظرة عامة
تطبيق **صيدلية يمان** هو نظام نقاط بيع (POS) متكامل للصيدليات، يتكون من صفحتين رئيسيتين:
1. **الصفحة الرئيسية** (`index.html`) - واجهة البيع للموظف
2. **لوحة التحكم** (`admin.html`) - إدارة النظام للمسؤول

---

## 📄 1. الصفحة الرئيسية (index.html)

### أقسام الصفحة الرئيسية:

#### 1.1 الترويسة (Header)
| العنصر | الوصف |
|--------|-------|
| الشعار | صورة دائرية للشعار (`img/logo.jpg`) |
| اسم الصيدلية | "صيدلية يمان" |
| معلومات المستخدم | اسم الموظف "حذفذف" مع أيقونة المستخدم |

#### 1.2 شريط التبويبات (Tabs)
| التبويب | الأيقونة | الوظيفة |
|---------|----------|---------|
| المبيعات | shopping-cart | واجهة البيع الرئيسية |
| الرفوف | layers | عرض المخزون ثلاثي الأبعاد |
| AR | scan-line | الكاميرا المعززة |
| السجل | history | سجل المبيعات |
| الادارة | settings | الانتقال لصفحة الإدارة |

#### 1.3 قسم المبيعات (Sales View)

| المكون | الوصف |
|--------|-------|
| **لمحة AI** | فقاعة معلومات تظهر وصف الدواء المحدد |
| **صندوق البحث** | حقل بحث مع قائمة منسدلة لاختيار الأدوية |
| **زر الباركود** | زر تشغيل ماسح الباركود |
| **اختيار الوحدة** | أزرار: باكت / علبة / شريط / حقنة / مخصص |
| **تحديد الكمية** | أزرار + و - مع حقل إدخال الرقم |
| **عرض الأسعار** | سعر الوحدة + إجمالي المنتج |
| **زر الإضافة** | إضافة منتج آخر للفاتورة |
| **الفاتورة** | جدول يعرض الأصناف المضافة مع الإجمالي |
| **زر الطباعة** | إصدار وطباعة الفاتورة |

**قاعدة بيانات الأدوية المدمجة:**
- Amoxicillin 500mg
- Panadol Advance
- Ibuprofen 400mg
- Nexium 40mg
- Aspirin Protect
- Vitamin C 1000mg

#### 1.4 قسم الرفوف (Inventory View)

| المكون | الوصف |
|--------|-------|
| **المشهد ثلاثي الأبعاد** | 3 جدران (600×400 بكسل) كل جدار 300 خلية (20×15) |
| **خلايا المخزون** | كل خلية تمثل موقع دواء مع إحداثيات (جدار-صف-عمود) |
| **الحالات اللونية** | أخضر: طبيعي / أصفر: قرب انتهاء الصلاحية / أحمر: مخزون منخفض |
| **نافذة المنبثقة** | عرض تفاصيل الأدوية عند النقر على الخلية |

#### 1.5 قسم AR (Augmented Reality)

| المكون | الوصف |
|--------|-------|
| **SkyView** | محاكاة الكاميرا للتعرف على المخزون |
| **التسميات المعززة** | عرض كميات الأدوية فوق الرفوف الحقيقية |
| **تنبيهات المخزون** | إشعار عند انخفاض الكمية |

#### 1.6 قسم السجل (Recent View)

| المكون | الوصف |
|--------|-------|
| **قائمة الفواتير** | عرض فواتير اليوم مع الرقم والوقت والقيمة |
| **تفاصيل العملية** | عدد الأصناف والموظف المنفذ |

## 📄 2. لوحة التحكم (admin.html)

### أقسام لوحة التحكم:

#### 2.1 الترويسة (Header)
- الشعار + "لوحة التحكم - يمان"
- زر الرجوع للرئيسية

#### 2.2 شريط التبويبات
| التبويب | الأيقونة | الوظيفة |
|---------|----------|---------|
| التحليلات | bar-chart-3 | إحصائيات المبيعات |
| المخزون | box | إدارة المشتريات والمخزون |
| المالية | wallet | المتتبع المالي والتقارير |
| الإعدادات | settings | إعدادات النظام |

#### 2.3 قسم التحليلات (Analytics)

| المكون | الوصف |
|--------|-------|
| **بطاقات الإحصائيات** | إجمالي المبيعات اليومية + عدد الفواتير |
| **الرسم البياني** | SVG يوضح أداء المبيعات الأسبوعي |
| **جدول الفواتير** | تفاصيل الفواتير الصادرة (رقم - أصناف - قيمة - وقت - موظف) |

#### 2.4 قسم المخزون (Inventory Admin)

| المكون | الوصف |
|--------|-------|
| **نموذج الإضافة** | إدخال: اسم الدواء / الكمية / الموقع (W1-R5-C10) |
| **زر الحفظ** | حفظ في المخزون |
| **مفتاح العرض** | تبديل بين 3D و شبكة عالية |
| **عرض 3D** | 3 جدران أصغر (300×200) |
| **الشبكة العالية** | عرض تفصيلي HD للمخزون |
| **ماسح AI** | محاكاة كاميرا للتدقيق على المخزون |

#### 2.5 قسم المالية (Finance)

| المكون | الوصف |
|--------|-------|
| **أزرار الإضافة** | إضافة دخل / إضافة نفقة |
| **نموذج العملية** | نوع العملية (إيجار/كهرباء/رواتب/أخرى) + المبلغ |
| **حاسبة الربح** | إجمالي المبيعات - إجمالي النفقات = صافي الربح |
| **التقارير** | إحصائيات يومية / أسبوعية / شهرية / سنوية |

#### 2.6 قسم الإعدادات (Settings)

| المكون | الوصف |
|--------|-------|
| **العلامة التجارية** | اسم الصيدلية + تحميل الشعار |
| **إدارة الموظفين** | قائمة الموظفين مع الصلاحيات |
| **الإعدادات التقنية** | حالة الطابعة + النسخ الاحتياطي السحابي |

- **تصميم داكن** (Dark Mode) بألوان أزرق-تركواز
- **تصميم متجاوب** يتكيف مع الجوال وسطح المكتب
- **في سطح المكتب**: الشاشة مقسمة إلى نصفين (65% عرض ديناميكي / 35% مبيعات ثابتة)
